# todo
Todo app - REST


# environment
- python3
- django
- django-rest-framework
- django-filter


# how to run
 - git clone git@github.com:dhruvsingh/todo.git
 - make a virtual environment (ensure it is python3)
 - activate virtual environment
 - cd todo/todo
 - ./manage.py migrate
 - ./manage.py runserver
 - browse to http://127.0.0.1/api/task/ to view list of all tasks
